# **Important!**
In order to make this shell script work, you need to install the following add-ons:

- iStats
- lolcat
- fortune
- artii
- cowsay

### [iStats](https://github.com/Chris911/iStats)
Installation
```
$ gem install iStats
```

###lolcat
Installation
```
$ gem install lolcat
```
###fortune
Installation
```
$ brew install fortune
```
###artii
Installation
```
$ gem install artii
```
###cowsay
Installation
```
$ brew install cowsay
```
